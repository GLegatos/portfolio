Super Image Brain

Super Image Brain is an app with a front built in React and a back end built with Node.js and Express.js. When a new user registers, their information is stored in a PostgreSQL Database. Users can then take any image URL and the Super Image Brain will detect the subject of the image using an image detection API. The image counter will also keep track of a User's entires eveytime they log in and out.
